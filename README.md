# Restaurant reviews Web app

## To run 

cd restaurants   
npm i                                

cd server    
npm i 


cd restaurants 
npm i 

cd server 
npm i 


cd .. 
(we are back into 'restaurants' directory)   

npm start

Visit the site at:
 http://localhost:8000/
