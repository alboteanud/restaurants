# Restaurant reviews Web app

## To run 

cd restaurants   
npm i                                

cd server    
npm i 


cd .. 
(we are inside 'restaurants' folder)   

npm start

Visit the site at:
 http://localhost:8000/  



## Notes:

- Disable Internet Access completely in order to test "put review while offline" 
(Dev tools offline will not be enough - using workbox)
